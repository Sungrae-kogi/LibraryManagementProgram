제목 : 도서 대여 프로그램 개발

* 전제 조건 : 학원 등 학습한 내용 기반으로 개발하여야 하며 공개된 소스를 활용하면 안됨


* 개요
   - 도서관에서 도서를 관리하는 프로그램을 개발하는 것이 목표임
   - 1단계로 1인의 관리자가 조회, 입력, 대출, 반납 을 처리하는 것을 가정하되 추후 관리자가 추가될 경우 각 관리자의 작업 내역을 구분할 수 있도록 확장성을 갖도록 개발하여야 함
   - 프로그램 실행 각 단계별 동작은 상식선에서 동작되어야 함
   - 데이터 타입, 테이블의 구성 등은 가장 적합하다고 생각되는 것으로 선정하여 사용할 것
   - 각각의 항목은 데이터 정합성이 맞아야 함 (예를 들어 전체 도서 중 대여 중인 것과 반납된 내역의 정합성 등)
   - DB는 Oracle, 언어는 Java 사용, WAS및 WebServer는 자유롭게 선택


* 상세 내용

1. 관리자 1명이 사용하는 프로그램을 가정하여 개발   
  - 관리자 1인 외 여러 사용자가 추가될 것을 염두에 두고 확장가능하게 개발

2. 도서 입력
  2-1. 도서 테이블을 생성(도서 번호, 도서명, 저자, 수서날짜, 발행일자, ISBN 등 기타 필요하다고 생각되는 정보를 선택하여 필수 입력과 선택 입력 구분하여 개발) 
  2-2. 복본(동일 책이 여러권이 있는 것) 처리 할 것
  2-3. 도서 조회,입력,수정,삭제 등 기본 기능 개발
  2-4. 각 도서의 상태 및 이력을 조회할 수 있어야 함
  2-5. 샘플 데이터는 복본이 없는 도서 5권, 복본이 2개인 도서 5권, 복본이 3개인 도서 5권 으로 구성하되 실제 있는 도서를 입력할 것

3. 도서 대여
  3-1. 대여 테이블 생성(도서 번호, 대여자의 교번(학번), 이름, 대여 상태 구분, 대출일, 반납예정일, 반납일 등 도서 대여와 관련된 내역)
  3-2. 도서 상태에 맞게 조회,입력,수정,삭제 등의 동작이 이루어져야 함
  3-3. 현재 대여중인 도서 목록을 확인할 수 있어야 함
  3-4. 반납기간이 경과한 도서를 검색할 수 있어야 함

4. 명시하지 않은 기능은 프로그램 목적에 맞게 가감하여 구성해도 무방하며 새로고침, 화면이동 등을 주의하여 개발
